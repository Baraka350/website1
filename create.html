What is HTML?

Definition of HTML

HTML stands for Hypertext Markup Language. It is the standard language used to create and design web pages. HTML provides the structure for web content, including elements like headings, paragraphs, links, and images.

Who Created HTML?

HTML was created by Tim Berners-Lee, a British computer scientist, in 1991 while working at CERN (European Organization for Nuclear Research).

HTML as a Markup Language

HTML is a markup language, which means it uses tags to structure and format content. Unlike programming languages that are used to create logic or functionality, markup languages like HTML define the structure and presentation of data on a webpage.

What is a Markup Language? (5 Key Points)

  • Markup languages are used to annotate or structure content on a page.
  • They use tags or symbols to define the structure and presentation of text and multimedia elements.
  • They are not programming languages and do not define logic or functionality.
  • Markup languages make it easy to represent data in a readable and formatted way on the web.
  • HTML, XML, and Markdown are examples of markup languages.

Basic Structure of HTML

The basic structure of an HTML document includes the following essential elements:

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>Page Title</title>
    </head>
    <body>
        <h1>This is a Heading</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

Attributes and Types of Tags

HTML tags can have attributes that define specific properties or behaviors of the element. Attributes are placed inside the opening tag and are written as key-value pairs.

Common Types of HTML Tags:

  • Structural Tags: Used for defining the structure of the webpage.
  • Text Formatting Tags: Used to format text on the page.
  • Anchor Tags: Used for creating hyperlinks.
  • Image Tags: Used for embedding images.
  • List Tags: Used to create ordered or unordered lists.
  • Table Tags: Used for creating tables.
  • Form Tags: Used for collecting user input through forms.
  • Meta Tags: Used for providing metadata about the webpage.
  • Div and Span Tags: Used for grouping elements.
