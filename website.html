I.C.1: Description of Keyboard and Its Use

A computer keyboard is an input device used to type text, execute commands, and interact with a computer system. It consists of multiple keys, each serving different functions such as letters, numbers, symbols, and special commands.

Types of Keyboards Based on Key Layout

  • QWERTY Keyboard: The most common layout, named after the first six letters on the top row.
  • AZERTY Keyboard: Used mainly in French-speaking countries.

Types of Keyboards Based on Communication with a Computer

  • Wired Keyboard: Connects to a computer using a USB or PS/2 cable, providing a stable connection.
  • Wireless Keyboard: Uses Bluetooth or RF technology to connect, offering portability and convenience.
  • Virtual Keyboard: A software-based keyboard displayed on a touchscreen or accessed remotely.

Parts of a Computer Keyboard

  • Alphanumeric Keys: These keys consist of letters (A-Z) and numbers (0-9), used for typing text and entering data.
  • Function Keys (F1-F12): Located at the top of the keyboard, they are used for various functions like opening help windows, refreshing pages, or executing specific commands depending on the software.
  • Control Keys: These include the Ctrl, Alt, and Shift keys, which are used in combination with other keys for shortcuts and commands. For example, Ctrl + C for copy.
  • Navigation Keys: These keys include the Arrow keys, Home, End, Page Up, and Page Down. They help you navigate within documents, web pages, or programs.
  • Numeric Keypad: A section to the right of the main keyboard that contains numbers and mathematical symbols (addition, subtraction, multiplication, division) for easier data entry and calculations.
  • Special Keys: Includes keys like Enter (used to submit commands or confirm actions), Backspace (removes characters), Delete (removes items or characters), Escape (Esc) (used to exit or cancel operations), and the Windows key (opens the Start menu in Windows).
